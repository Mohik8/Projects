---

## Algorithms

| Algorithm | Modes | Notes |
|-----------|-------|-------|
| Caesar Cipher | encrypt / decrypt / brute-force | ROT-N with full 25-shift scanner |
| VigenГЁre Cipher | encrypt / decrypt | Polyalphabetic key stream |
| XOR Stream Cipher | encrypt / decrypt | Hex key, arbitrary length |
| AES-128 | ECB + CBC encrypt/decrypt | Full FIPS 197 вҖ” S-box, ShiftRows, MixColumns, KeySchedule from scratch |
| Base64 | encode / decode | Handles padding |

---

## Build

Requires GCC (or any C11-compliant compiler).

```sh
gcc -Wall -O2 -std=c11 src/main.c src/aes.c src/ciphers.c src/base64.c src/utils.c src/visuals.c -o cryptkit
```

Or use Make:

```sh
make
```
